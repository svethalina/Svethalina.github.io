Projects

YOLO

Recognition of Handwritten Digits using AI

Developed an ANN model for handwritten digit recognition using the MNIST dataset. Achieved high accuracy in classifying tens of thousands of handwritten digit images. Saved the trained model in HDF5 format for easy deployment.

Sentimental Analysis on Movie Reviews using AIML

Developed a machine learning model for sentiment analysis of movie reviews using Python, scikit-learn, and NLTK. Preprocessed text data, extracted TF-IDF features, and trained Naive Bayes and Logistic Regression classifiers to predict positive, negative, or neutral sentiment with up to 88% accuracy.

Fatigue Level Detection using Heart rate Sensor

Developed a system to detect fatigue levels while using computer devices like mouse and keyboard by analyzing heart rate data. Implemented an Artificial Neural Network model using TensorFlow to train on the collected heart rate measurements and classify stress levels with 78.6% accuracy. Addressed issues like repetitive strain injury and carpal tunnel syndrome caused by prolonged, monotonous computer usage.

Student Behavior Analysis using Unsupervised approach

Developed a machine learning system to analyze student performance based on demographic data, study habits, and psychological factors using unsupervised learning techniques. Implemented data preprocessing, feature extraction, and evaluation models achieving 79% accuracy in classifying students into high, average, and low performance categories.

Touch Screen and RFID based smart hotel menu system

Using IoT, engineered an innovative restaurant order system utilizing Arduino UNO microcontroller, RFID for secure payments, and LCD display for interactive menu engagement; facilitated TCP Telnet Terminal App configuration for seamless IoT integration, elevating customer service quality.
